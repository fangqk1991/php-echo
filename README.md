# php-echo

### Getting Started

Add php-request to your application's composer.json.

```
{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/fangqk1991/php-echo"
    }
  ],
  ...
  ...
  "require": {
    "fang/php-echo": "dev-master"
  }
}

```

Run `composer install`
